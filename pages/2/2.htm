C++ - Настройка рабочей среды

Если вы по-прежнему готовы настроить свою среду для C ++, вам необходимо иметь следующие два программного обеспечения на вашем компьютере.

Текстовый редактор

Это будет использоваться для ввода вашей программы. Примеры нескольких редакторов включают в себя: Windows Notepad, OS Edit command, Brief, Epsilon, EMACS, and vim or vi.

Имя и версия текстового редактора могут различаться в разных операционных системах. Например, Notepad будет использоваться в Windows, а vim или vi можно использовать как в Windows, так и в Linux или UNIX.

Файлы, созданные вами в редакторе, называются исходными файлами, а для C ++ они обычно называются с расширением .cpp, .cp или .c.

Текстовый редактор должен быть на месте, чтобы начать программирование на C ++.

Компилятор C ++

Это фактический компилятор C ++, который будет использоваться для компиляции исходного кода в конечную исполняемую программу. Большинство компиляторов C ++ не заботятся о том, какое расширение вы передаете исходному коду, но если вы не укажете иное, многие будут использовать .cpp по умолчанию.

Наиболее часто используемым и бесплатным компилятором является компилятор GNU C / C ++, иначе вы можете иметь компиляторы либо из HP, либо из Solaris, если у вас есть соответствующие операционные системы.

Установка компилятора GNU C / C ++

Установка UNIX / Linux

Если вы используете Linux или UNIX, тогда проверьте, установлен ли GCC в вашей системе, введя в командной строке следующую команду:

$ g++ -v

Если вы установили GCC, тогда он должен напечатать сообщение, например следующее:

Using built-in specs.
Target: i386-redhat-linux
Configured with: ../configure --prefix=/usr .......
Thread model: posix

Если GCC не установлен, вам необходимо будет установить его самостоятельно, используя подробные инструкции, доступные на странице https://gcc.gnu.org/install/

Установка Mac OS X

Если вы используете Mac OS X, самый простой способ получить GCC - загрузить среду разработки Xcode с веб-сайта Apple и следовать простым инструкциям по установке.

Xcode в настоящее время доступен на developer.apple.com/technologies/tools/ .

Установка Windows

Чтобы установить GCC на Windows, вам необходимо установить MinGW. Чтобы установить MinGW, перейдите на домашнюю страницу MinGW, www.mingw.org и перейдите по ссылке на страницу загрузки MinGW. Загрузите последнюю версию программы установки MinGW, которую следует назвать MinGW- <версия> .exe.

При установке MinGW, как минимум, вы должны установить gcc-core, gcc-g ++, binutils и среду исполнения MinGW, но вы можете установить больше.

Добавьте подкаталог bin вашей установки MinGW в переменную среды PATH, чтобы вы могли указать эти инструменты в командной строке своими простыми именами.

По завершении установки вы сможете запустить gcc, g ++, ar, ranlib, dlltool и несколько других инструментов GNU из командной строки Windows.
